bash的简单扩展
=====
1.配置说明
----------
    1.初始化
```sh
    $ cd xbash
    $ sudo -k chmod -R a+x module/ config/ init
    $ ./init
```
    2. 查看使用说明
```sh
    $ xb -h
```

3.其他说明
----------
    01 为了方便偷懒，部分实现在方法实现之外又加了一层命令简化封装，所以一种功能可能存在两种调用方式。
    02 命令简化封装和部分方法命名不符合规范,属于历史遗留和个人使用习惯问题
    03 用户模块[用户名.module]中添加配置会默认覆盖原始配置
    04 用户模块下定义会覆盖同名的默认定义
